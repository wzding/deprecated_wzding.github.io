COVID-19 Visualization in the U.S.

This narrative visualization demonstrates State-level comparisons for COVID-19 cases and deaths. The 1st graph shows the time series of COVID-19 cases from Feb. 2020 to Jul. 2020. The 2nd visualization presents the number of cases and deaths in each State sorted by the number of deaths in descending order. Each month is color coded as a different color. The last scatter plot demonstrates the ratio of deaths to cases. Red circles show the highest ratios while blue ones indicate lowest ratios.
